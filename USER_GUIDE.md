# AI Editorial Layout Agent - 用户指南

## 快速开始

### 1. 安装

```bash
# 克隆项目
cd editorial_agent

# 安装依赖
pip install -r requirements.txt
```

### 2. 启动应用

```bash
# 方式1：使用启动脚本
bash start.sh

# 方式2：直接运行
streamlit run app.py
```

### 3. 访问应用

打开浏览器访问：`http://localhost:8501`

## 使用流程

### 第一步：输入文章链接

在侧边栏的"文章链接"输入框中粘贴微信公众号文章链接：

```
https://mp.weixin.qq.com/s/xxxxxxxxxxxx
```

### 第二步：配置选项

#### 卡片数量 (5-18张)
- 默认：13张
- 根据文章内容长度调整
- 短篇内容建议5-8张
- 长篇内容建议10-18张

#### 图片尺寸
| 尺寸 | 比例 | 适用场景 |
|------|------|----------|
| 1080×1350 | 4:5 | Instagram帖子 |
| 1080×1080 | 1:1 | 方形图片 |
| 1080×1920 | 9:16 | Instagram Stories |
| 1200×1500 | 4:5 | 大尺寸打印 |
| 1200×1200 | 1:1 | 大尺寸方形 |

#### 视觉风格
| 风格 | 特点 | 适用场景 |
|------|------|----------|
| Editorial | 杂志编辑风格，优雅经典 | 专业内容 |
| Minimal | 简洁干净，现代感强 | 极简主义 |
| Warm | 暖色调，温馨舒适 | 生活方式 |
| Dark | 深色背景，高端大气 | 夜景/高端 |
| Pastel | 柔和色彩，清新淡雅 | 女性/甜点 |

### 第三步：生成卡片

点击"生成图文卡片"按钮，系统将自动：
1. 抓取文章内容
2. 解析内容结构
3. 匹配图片素材
4. 生成排版布局

### 第四步：预览与调整

#### 浏览卡片
- 使用"上一张"/"下一张"按钮切换
- 点击缩略图快速跳转
- 查看布局类型和文字内容

#### 替换图片
- 点击"替换当前图片"按钮
- 系统会重新搜索匹配图片
- 保持排版结构不变

### 第五步：导出文件

点击"生成 PPTX 文件"按钮：
- 生成可编辑的PowerPoint文件
- 所有文字和图片可修改
- 保持独立分层结构

## 布局类型说明

### 1. Full Image + Bottom Text
- 图片占70%，文字占30%
- 适合：产品展示、成品图片

### 2. Left Image + Right Text
- 左右各50%分割
- 适合：对比展示、步骤说明

### 3. Top Text + Bottom Image
- 文字在上，图片在下
- 适合：说明性内容、配方展示

### 4. Center Title (Minimal)
- 纯文字居中
- 适合：章节标题、引言

### 5. Two Image Collage
- 两张图片并排
- 适合：多角度展示、对比

### 6. Half Image + Half Text
- 半图半文字，无内边距
- 适合：视觉冲击力强

### 7. Text Only (Minimal)
- 仅文字内容
- 适合：引用、总结、心得

### 8. Overlay Text on Image
- 文字叠加在图片上
- 适合：海报风格、强调文字

### 9. Three Image Grid
- 三张图片网格排列
- 适合：步骤展示、系列作品

### 10. Full Bleed Image
- 图片铺满整张卡片
- 适合：封面、结尾、视觉冲击

## 常见问题

### Q1: 文章抓取失败怎么办？

**可能原因：**
- 文章链接无效
- 文章需要登录访问
- 网络连接问题

**解决方案：**
- 检查链接是否正确
- 尝试在浏览器中打开链接
- 检查网络连接

### Q2: 图片无法显示怎么办？

**可能原因：**
- 原文图片已删除
- 图片链接失效
- 网络访问限制

**解决方案：**
- 使用"替换当前图片"功能
- 检查网络连接
- 手动替换为本地图片

### Q3: 如何调整文字大小？

**方法1：** 在额外需求中设置
```
font_size_title=48
font_size_body=28
```

**方法2：** 导出后在PowerPoint中手动调整

### Q4: 如何批量处理多篇文章？

目前版本支持单篇文章处理。批量处理功能将在后续版本中添加。

### Q5: 导出的PPTX文件打不开？

**可能原因：**
- 文件损坏
- PowerPoint版本过低

**解决方案：**
- 重新生成文件
- 使用Microsoft PowerPoint 2016或更高版本
- 尝试使用WPS Office打开

## 最佳实践

### 内容优化
1. **标题简洁**：确保文章标题清晰明了
2. **段落分明**：使用小标题分隔内容
3. **图片质量**：原文图片分辨率越高越好

### 布局选择
1. **封面**：使用 Full Bleed Image 或 Full Image + Bottom Text
2. **章节标题**：使用 Center Title (Minimal)
3. **正文内容**：轮换使用不同布局
4. **结尾**：使用 Full Bleed Image 或 Text Only

### 视觉风格
1. **品牌一致性**：选择与品牌调性相符的主题
2. **内容匹配**：烘焙类内容适合 Warm 主题
3. **季节变化**：根据季节调整色彩
