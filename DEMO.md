# AI Editorial Layout Agent - 功能演示

## 系统概述

AI Editorial Layout Agent 是一个智能的内容排版系统，能够将微信公众号长文章自动转化为视觉化的图文卡片，适用于社交媒体发布。

## 核心功能演示

### 1. 文章抓取与解析

**输入**：微信公众号文章链接
```
https://mp.weixin.qq.com/s/xxxxx
```

**处理流程**：
1. 发送HTTP请求获取文章HTML
2. 使用BeautifulSoup解析页面结构
3. 提取标题、作者、发布时间、正文和图片
4. 识别文章的小标题层级

**输出示例**：
```json
{
  "title": "如何制作完美的法式可颂",
  "author": "烘焙小课堂",
  "publish_time": "2024-01-15",
  "content_text": "...",
  "original_images": [
    {"url": "...", "caption": "..."}
  ],
  "sections": [
    {"level": 1, "text": "准备材料"},
    {"level": 1, "text": "制作步骤"},
    {"level": 1, "text": "烘焙技巧"}
  ]
}
```

### 2. 语义拆解

**输入**：完整文章内容

**处理逻辑**：
- 识别小标题作为分割点
- 保持段落语义完整性
- 自动调整块数量以匹配目标卡片数

**输出**：内容块列表

| 块序号 | 标题 | 类型 | 字数 |
|--------|------|------|------|
| 1 | 如何制作完美的法式可颂 | intro | 45 |
| 2 | 准备材料 | section | 120 |
| 3 | 制作步骤 | section | 280 |
| 4 | 烘焙技巧 | section | 150 |
| 5 | 总结 | content | 80 |

### 3. 布局生成

**10种排版布局自动轮换**：

1. **大图配底部文字** (full_image_bottom_text)
   - 图片占70%，文字占30%
   - 适合展示产品图片

2. **居中标题留白** (center_title_minimal)
   - 纯文字，居中显示
   - 适合章节标题页

3. **左图右文** (left_image_right_text)
   - 左右各50%分割
   - 适合对比展示

4. **上文下图** (top_text_bottom_image)
   - 文字在上，图片在下
   - 适合说明性内容

5. **双图拼贴** (two_image_collage)
   - 两张图片并排
   - 适合展示多个角度

6. **纯文字极简** (text_only_minimal)
   - 仅文字内容
   - 适合引用或总结

7. **半图半文字** (half_image_half_text)
   - 无内边距分割
   - 视觉冲击力强

8. **图片上叠加文字** (overlay_text_on_image)
   - 文字叠加在图片上
   - 适合海报风格

9. **三图网格** (three_image_grid)
   - 三张图片网格排列
   - 适合步骤展示

10. **全出血图片** (full_bleed_image)
    - 图片铺满整张卡片
    - 适合封面或结尾

### 4. 图片匹配

**三级图片获取策略**：

**优先级1：原文图片**
- 使用文章中原有图片
- 根据标题/说明匹配相关性

**优先级2：Instagram图片**
- 从指定Instagram账号获取
- 保持视觉风格一致

**优先级3：互联网搜索**
- 使用Unsplash等免费图库
- 基于内容关键词搜索

### 5. 视觉风格

**5种预设主题**：

| 主题 | 背景色 | 文字色 | 强调色 | 字体 |
|------|--------|--------|--------|------|
| Editorial | #FAFAFA | #2C2C2C | #C9A961 | Times New Roman |
| Minimal | #FFFFFF | #1A1A1A | #E8E8E8 | Helvetica Neue |
| Warm | #FDF8F3 | #4A3728 | #D4A574 | Georgia |
| Dark | #1A1A1A | #F5F5F5 | #333333 | Helvetica Neue |
| Pastel | #F5F0F5 | #4A4A4A | #E8D5E8 | Helvetica Neue |

### 6. PPTX导出

**导出特性**：
- 所有文字可编辑
- 所有图片可替换
- 独立分层结构
- 支持多种尺寸

**支持的尺寸**：
- 1080×1350 (4:5 Instagram)
- 1080×1080 (1:1 方形)
- 1080×1920 (9:16 故事)
- 1200×1500 (4:5 大尺寸)
- 1200×1200 (1:1 大尺寸)

## 技术栈

- **Python 3.8+**
- **Streamlit** - Web界面框架
- **BeautifulSoup4** - HTML解析
- **python-pptx** - PPTX生成
- **Pillow** - 图片处理
- **Requests** - HTTP请求

## 性能优化

- **图片缓存**：下载的图片本地缓存，避免重复下载
- **异步处理**：文章抓取和图片下载可并行处理
- **增量更新**：仅更新修改的卡片，不重新生成全部

## 注意事项

1. **微信公众号限制**：部分文章可能需要登录
2. **图片版权**：注意图片使用授权
3. **网络依赖**：图片搜索需要网络连接
4. **内存管理**：大量图片处理时注意内存使用
